# ECCベストワン プロポーザル作成ツール (Prototype)

保護者向けの魅力的な学習提案書を作成する内部ツールです。
Gemini API を利用し、簡単な質問に答えるだけで高密度なプロポーザルを自動生成します。

## 使い方

1. **APIキーの準備**:
   - Google Gemini APIキーが必要です（[Google AI Studio](https://aistudio.google.com/)等で取得）。

2. **ツールの起動**:
   - `index.html` をブラウザで開きます。
   - ※注意: ES Modulesを使用しているため、単にファイルをダブルクリックするだけでは動作しない場合があります（CORSエラー等）。
   - VS Codeの「Live Server」拡張機能を使用するか、以下のコマンドでローカルサーバーを立ち上げてアクセスしてください。
     ```bash
     # Pythonがインストールされている場合
     python -m http.server
     # その後、ブラウザで http://localhost:8000 にアクセス
     ```

3. **操作手順**:
   - 画面右上の入力欄にAPIキーを入力し「保存」します（初回のみ）。
   - **教室選択**: 藍住 または 北島中央 を選択。
   - **対象生徒**: 新規 / 在校 / 英会話 から選択。
   - 「質問を開始」ボタンを押し、表示されるフォームに入力します。
   - 「プロポーザル生成」ボタンを押すと、AIが内容を生成します。
   - プレビューを確認し、問題なければ「PDFとして保存/印刷」ボタンで印刷ダイアログを開き、「PDFに保存」を選択してください。

## カスタマイズについて
- `style.css`: 色やレイアウトの調整。
- `script.js`: 質問項目の追加や、AIへのプロンプト（指示）の微調整。

## 新機能: PDFチラシの管理 (V28以降)
提案書作成画面に、関連するPDFチラシへのリンクを表示できます。

1. **PDFの追加**: プロジェクトフォルダ内の `pdf` フォルダにPDFファイルを保存します。
2. **リスト更新**: `update_pdf_list.bat` をダブルクリックして実行します。
3. **確認**: ブラウザをリロードすると、新しいファイルが表示されます。
   （この仕組みにより、Vercel等のWebサーバー上でもPDFリスト機能が動作します）

## その他の便利機能
- **自動保存**: 入力内容は自動的にブラウザに保存され、閉じても消えません。
- **全クリア**: フォーム下部のボタンで、保存された入力内容を全てリセットできます。
